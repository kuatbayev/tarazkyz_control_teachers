# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/b51a67a9-4eec-4b53-84d1-37b3fb2b622b

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Configure the required environment variables in [.env.local](.env.local)
3. Run the app:
   `npm run dev`
